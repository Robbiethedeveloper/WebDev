NOTES
Padding - refers to the space between an element's content and its border.
Margin - refers to the additional space or area around or outside an element.
First element inside a section element should be a heading
Main element should only be used once per page and not on repeating information such as a nav bar, etc.  Essentially, use for stuff unique to that page.
Div element has no semantic meaning.  It's just a wrapper that groups together related content.  
Can use multiple footers and headers on a single page.
The image element is an empty element.  This means it doesn't contain any child content and it doesn't have a closing tag.
Empty elements, such as img, hr, and br can have a trailing slash and this is to indicate that it is an empty element.  This is optional.
You shouldn't use the hr element to have a line and you should use 3 or more line breaks typically.  Athestics are best handled by CSS
Just like div, when there is no suitable to differentiate a piece of text from surrounding text, use span.  Both Div and Span both have no semantic meaning on its own.
